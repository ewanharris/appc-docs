Studio 4.3.3 Release Note

Studio 4.3.3 - 27 October 2015

Studio 4.3.3 is a patch release addressing high-priority issues from previous releases.

Fixed Issues

  • TISTUD-7747: LiveView does not work: Script Error Couldn't find module: localeStrings for architecture: x86_64

  • TISTUD-7780: Launching simulator/emulator creates a new Node.js process without killing the previous one

  • TISTUD-7791: Studio prompts for update to incorrect NPM Appc CLI and SDK

  • TISTUD-7796: If you run "npm ls -g appcelerator" when a pre-release version is installed, "npm ERR! code 1" is returned
