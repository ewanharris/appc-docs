If you created a Arrow application and associated it with your organization, or the client application makes requests to an Arrow application using the Titanium HTTP client, you can view information about the Arrow application in Dashboard. To associate a Arrow application with a client application built with Java or Objective-C, you need to add x-native-id header to the HTTP request as instructed in Associate Arrow Applications with Client Applications.

Arrow API Calls Metric Details Screen

To view aggregate data about all your organization's Arrow applications, from the Dashboard Home screen or the Metric Details screen, click on the Metric Details drop-down and select Cloud API Calls (Custom).

The Cloud API Calls (Custom) metric details page opens. This page displays data about all of your organization's Arrow applications, such as total API calls to the application, CPU usage, memory usage and number of Titanium client applications making calls to the Arrow application.

Click on a Service Name to view more details about the Arrow application.

Arrow Application Details Screen

When you select a Arrow application in either the client application's Cloud Custom Service tab or Cloud API Calls (Custom) metric details page, its details are displayed in a new page. From the details screen, you can view basic information about the application and a log of API calls made to the application.

Overview

Click on the Overview button to view basic information about the application. This page displays:

  • The endpoint URL to access the application

  • Creation and last published date of the application

  • Total API calls made to the application

  • Number of cloud containers used by the application

  • Overall CPU and memory usage and also broken down by cloud containers

If you are an organization administrator, you can also see a list of the connected client applications using the Arrow application. Click on the application name to open the application's Cloud Custom Services tab.

Logs

Click the Logs button to display a list of API calls made to the application. Each entry in the log lists the URL path, response time, remote IP that made the call, the cloud container that handled the call, and the datetime of the request.

Viewing Arrow Applications Used by a Client Application

From the Application Details screen, you can view the Arrow applications the client application is using. To view the list:

  1. Select the application from the Apps menu.

  2. Click the Cloud tab.

  3. Click the Custom Services sub-tab.

Click on a Service Name to view more details about the Arrow application.
