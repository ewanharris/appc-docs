Alloy 1.7.34 Release Note

Alloy 1.7.34 - 24 February 2016

Alloy 1.7.34 is a patch release that includes new features, improvements and bug fixes.

New Features

  • ALOY-1356 and ALOY-1357 - Support for defining the Ti.UI.iOS.LivePhotoView and Ti.UI.iOS.MenuPopup in XML

Improvements

  • ALOY-1352 - The methods dpToPX and pxToDP of the measurement module support HD Retina iOS

Fixed Issues

  • ALOY-1297 - Cannot use localized strings for AlertDialog ButtonNames in XML

  • ALOY-1355 - Alloy doesn't cast "true"/"false" attributes as booleans
