Studio 4.3.1 Release Note

Studio 4.3.1 - 1 October 2015

Studio 4.3.1 is a patch release addressing high-priority issues from previous releases.

Fixed Issues

  • TISTUD-7732: NullPointerException when trying to Configure SDK

  • TISTUD-7726: Studio throws NPE while trying to save orgs

  • TISTUD-7725: Studio defaults to Beta Update stream for GA builds

  • TISTUD-7719: No simulators shown in studio with latest release

  • TISTUD-7710: Studio should handle the incompatible cache of ti info output

  • TISTUD-7708: Watchos2 app section is getting disappeared from tiapp editor after creating a new watchos2 app
