Alloy 1.8.5 Release Note

Alloy 1.8.5 - 29 March 2016

Alloy 1.8.5 is a patch release that includes improvements and bug fixes.

Improvements

  • ALOY-1281 - RSS sample should be removed in favour of Studio version

    • Replaced the Alloy RSS sample app with a Titanium Mobile RSS sample app

Fixed Issues

  • ALOY-1479 - Regression: Model instance data binding no longer works

  • ALOY-1486 - iOS, Android: Global styles is not overridden by local styles for orientationModes.

  • ALOY-1492 - Fix no such file or directory when doing "alloy new"

Known Issue

  • ALOY-1480 - Bug/Regression: Data binding always returns string values
