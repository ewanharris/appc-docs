Alloy 5.4.0.GA Release Note

Alloy 5.4.0.GA - 10 August 2016

Alloy 5.4.0.GA is a minor release that includes new features, improvements and bug fixes.

New Features

  • ALOY-1481 - Data binding: Support binding multiple models to the same view

    • Support binding multiple models to the same view

  • ALOY-1482 - Data binding: Support deep binding of model object attributes

    • Support deep binding of model attributes

  • ALOY-1494 - iOS: Support Ti.UI.iOS.Stepper in Alloy

    • Added support for the new Stepper UI element on iOS

Improvements

  • ALOY-1281 - RSS sample should be removed in favour of Studio version

    • Updated the RSS sample to reflect recent changes to Studio

Fixed Issues

  • ALOY-1478 - Data binding fails when binding string or model attribute name has dash, space or quote

  • ALOY-1480 - Bug/Regression: Data binding always returns string values

  • ALOY-1486 - iOS, Android: Global styles is not overridden by local styles for orientationModes.

  • ALOY-1503 - After Alloy compile, property which has array disappear.

  • ALOY-1508 - iOS: Localised splash screen is not visible
