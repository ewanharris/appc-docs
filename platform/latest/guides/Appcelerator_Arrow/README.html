Appcelerator Arrow

At a Glance

Appcelerator Arrow is an opinionated framework to build and deploy applications to the cloud.

Arrow Builder

Arrow Builder lets you build and deploy API endpoints that can be consumed by any client application. Arrow applications are made up of several components:

  • Arrow APIs are custom endpoints that allow you to access and execute custom operations on data.

  • Arrow Blocks are filters that allow you to pre- or post-process data.

  • Arrow Connectors are adaptors to allow you read and write data to and from an external data source, such as ArrowDB, MySQL, Salesforce, and MongoDB, or in server memory.

  • Arrow Models are structured data with predefined API endpoints to access them.

  • Arrow Web is a framework to create endpoints that render UI for HTML applications.

Relevant Topics: Arrow Quick Start, Arrow Project, Arrow API reference

Arrow Cloud

Arrow Cloud hosts applications, built with Arrow Builder, and Appcelerator pre-built services, which includes the ArrowDB and push notification services.

  • ArrowDB (formerly known as Appcelerator Cloud Services) provides pre-built REST objects, such as Users or Photos. ArrowDB can be integrated with your Titanium, Node.js, Android or iOS application.

  • Arrow Push (another former component of Appcelerator Cloud Services) provides the ability to send push notifications to Android and iOS devices subscribed to push channels. Arrow Push can be integrated with your Titanium, Android or iOS application.

Relevant Topics: Arrow Cloud, ArrowDB and Arrow Push
