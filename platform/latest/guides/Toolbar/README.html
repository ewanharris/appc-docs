Toolbar

Demonstrates adding a Toolbar control to an iOS application.

App Folder Location

alloy/test/apps/ui/toolbar

A Toolbar can contain Button controls and certain other controls, such as TextFields and Labels. To provide spacing between items in the toolbar, you can insert a <FlexSpace/> element to fill the space between items (as seen in the example) or a <FixedSpace/> element whose width attribute specifies the spacer width to add.

app/views/index.xml
<Alloy>
<Window>
<Toolbar platform="ios">
<!-- These get added to the Toolbar "items" array -->
<Items>
<Button>button 1</Button>
<FlexSpace/>
<Button>button 2</Button>
</Items>
</Toolbar>
<Label platform="android,mobileweb">Ti.UI.iOS.Toolbar only supported on iOS</Label>
</Window>
</Alloy>
