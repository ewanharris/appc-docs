Alloy 1.8.1 Release Note

Alloy 1.8.1 - 21 March 2016

Alloy 1.8.1 is a patch release that includes one new feature, one improvement and one bug fix.

New Feature

  • ALOY-1360 - Support for defining controller baseController in XML

Improvement

  • ALOY-1254 - Support model transform() in collection data-binding

Fixed Issue

  • ALOY-1369 - Complex data binding for collection doesn't work with transform
