Appcelerator CLI 5.5.0.GA Release Note

Appcelerator CLI 5.5.0.GA - 13 September 2016

Appcelerator CLI 5.5.0.GA is a minor release that includes two bug fixes.

Component Versions

The following components are shipped with CLI 5.5.0.GA:

Alloy

1.9.2

Arrow Builder

1.8.6

Arrow Cloud CLI

1.2.1

Titanium CLI

5.0.9

Fixed Issues

  • CLI-1070 - If you use node 0.12.X, then you'll be unable to download the Hyperloop module through 'appc new'

  • CLI-1079 - Modules and plugins are not overwritten if the files already exist
