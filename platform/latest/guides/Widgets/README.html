Widgets

Samples in the widgets/ folder demonstrates how to create and use Alloy widgets, which are self-contained Alloy components that can easily be shared.

  • Basic Widget — Demonstrates how to create and use a basic widget.

  • Complex Widget — Demonstrates a widget using another widget.

  • Widget Models — Demonstrates using data models with a widget.

  • WPATH() Example — Demonstrates how to use the WPATH() macro to easily map image and library paths to the widget folder.

You can find more Alloy widgets created by the community at gitt.io/.
