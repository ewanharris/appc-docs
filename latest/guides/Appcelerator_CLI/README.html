Appcelerator CLI

The Appcelerator CLI is a set of command-line tools to develop and deploy mobile and cloud applications. The Appclerator CLI provides a unified interface for using the other Appcelerator command-line tools, such as the Titanium CLI, Alloy CLI and Arrow Cloud CLI.
