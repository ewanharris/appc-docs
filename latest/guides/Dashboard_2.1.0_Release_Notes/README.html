Dashboard 2.1.0 Release Notes

The 2.1.0 release of Appcelerator Dashboard contains many bug fixes, as well as several new features and improvements.

  • New Features and Improvements

    • New Manage Orgs UI

    • New Status and Error pages

    • MessageGears Email Support

    • Feature Tour

    • Download Center: Insights and Enterprise Modules

    • Push Notification features

    • Other

  • Bug Fixes

New Features and Improvements

New Manage Orgs UI

The Manage Organizations user interface has been improved to make managing your organizations, applications, and members easier. For details see Managing Organizations.

New features:

  • Added ability to search for applications and members by name.

  • Added infinitely scrolling lists of applications and members.

  • A new Org Reporting tab contains widgets that display various metrics such as number of members an apps, and the number of pre-built and custom Cloud API calls.

New Status and Error pages

A new page at status.appcelerator.com reports the current status of all Platform services and applications. You can subscribe to receive status updates via email, RSS or web hook.

In addition, Dashboard error pages now provide more useful information, and provide links to the status page, open a feedback form, and contact technical support.

MessageGears Email Support

You can now configure Dashboard to use MessageGears to send email, in addition to SMTP.

Feature Tour

A new feature tour lets new users get up to speed with Dashboard features and functionality more quickly. To access the feature tour, click Getting Started in the bottom-left corner of Dashboard, then click Tour Dashboard.

Download Center: Insights and Enterprise Modules

You can now download Insights and Enterprise Modules for iOS or Android from the Dashboard Download Center.

Push Notification features

  • The iOS push configuration settings tab now displays the status (Enabled or Disabled) of your iOS push certificate.

  • Added the ability to unsubscribe a device from push notifications.

Other

  • Improved layout and selection highlight of Orgs and Apps drop-down menus.

  • Added pagination Analytics event tables.
